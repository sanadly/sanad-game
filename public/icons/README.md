# Icon Directory

This directory contains pixel art icons generated using Nano Banana.

## How to Generate Icons

1. Go to [Nano Banana](https://nanobanana.ai) or your preferred AI image generator
2. Use the prompts from `/lib/icons.ts` in the `nanoBananaPrompts` object
3. Generate 64x64 pixel images
4. Save them with the exact filenames specified in `iconPaths`
5. Place them in this directory

## Style Guidelines

All icons should:
- Be 64x64 pixels
- Use 16-bit pixel art style
- Have clean, transparent backgrounds
- Use vibrant, retro game colors
- Match the overall aesthetic of Terra Nova
